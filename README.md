# BrickPi

## Installation  
```bash
sudo raspi-config
```
```bash
sudo reboot
```
```bash
sudo apt install git python3-pip python3-rpi.gpio python3-smbus -y
```
```bash
git clone https://github.com/DexterInd/BrickPi3.git
```
```bash
sudo python3 setup.py install
```
